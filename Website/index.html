Software Engineer At Honeywell


At Honeywell, I worked across mission-critical systems, front-end development, and AI-driven solutions, contributing to multiple high-impact projects.

As part of the Flight Planning Engine (FPE) team, I developed a web-based platform that enabled commercial pilots to plan optimal flight routes by considering factors such as wind conditions, aircraft weight, and airspace restrictions. The system leveraged graph traversal algorithms for real-time route optimization and interfaced with ATC for automated route filing. I implemented this using C++, REST APIs, and React, ensuring seamless functionality and an intuitive user experience.

In the Reusable Web Components team, I contributed to creating modular, Honeywell-styled UI components, including search bars, buttons, and tables, to maintain a consistent design theme across various applications. Using PolymerJS and Node.js, I built scalable components that streamlined front-end development across teams, improving development efficiency and UI consistency.

I also worked on the Non-Technical Loss (NTL) project, which focused on electricity theft prevention using machine learning models. By analyzing usage patterns, the system could detect anomalous behavior and potential fraudulent customers. I developed and deployed these on-premise ML solutions using Python and machine learning SDKs, ensuring accurate and scalable fraud detection.
Through these projects, I gained hands-on experience in high-performance computing, AI-driven fraud detection, scalable UI development, and real-time optimization algorithms.
